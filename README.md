Retail Revenue Intelligence & Customer Segmentation Pipeline
Overview

This project presents an end-to-end retail analytics pipeline designed to analyze customer transaction data and extract business-critical insights.

The objective was to simulate a real-world e-commerce analytics workflow by calculating key performance metrics, identifying revenue drivers, segmenting customers, and visualizing trends for executive-level interpretation.

Dataset Summary

3,900 customer transactions

18 features including demographic, product, and behavioral attributes

Clean dataset (no missing values, no duplicates)

Key Performance Metrics

Total Revenue: $233,081

Average Order Value (AOV): $59.76

Total Unique Customers: 3,900

Business Insights
Revenue Drivers

Clothing category contributes approximately 45% of total revenue.

Male customers generate approximately 68% of overall revenue.

Demographic Segmentation

Age group 46–55 represents the highest spending segment.

Seasonal Trends

Fall season records the highest revenue performance.
